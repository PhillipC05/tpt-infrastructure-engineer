### How to run locally:

#### 1. __Install root dependencies__

```bash
npm install
```

#### 2. __Start Backend API__

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

✅ Backend will run at `http://localhost:8000` ✅ API Documentation: `http://localhost:8000/docs`

#### 3. __Start Frontend__

```bash
cd frontend
npm install
npm run dev
```

✅ Frontend will run at `http://localhost:5173`
